# プロジェクト概要
- 人生の残り時間・可処分時間を可視化するツール
- GitHub Pages でホスティング（`docs/` ディレクトリ）

# 技術制約
- plain HTML / CSS / JavaScript のみで実装
- フレームワーク・ライブラリの使用禁止（React, Vue, jQuery 等）
- ビルドツール不要（TypeScript, Sass, Webpack 等の変換処理なし）
- ブラウザネイティブAPIのみ使用

# 設計方針
- ビルドステップなしでそのままホスティングするだけでデプロイ可能なことが強み
- この特性を損なう変更（ビルド必須化、トランスパイル導入等）は避ける
